Mission / Vision / Value

全国展開を前提に、仕組みでスケールする。

Mission

未活用の「時間資産」を再設計し、都市に新たな価値を創出する。

空間ではなく“時間”に着目し、遊休を収益へ転換する仕組みを構築します。

Vision

日本中の遊休時間を、持続可能な収益源へ転換するインフラ企業になる。

店舗提携ネットワーク×運営標準化で、全国スケールの実装を行います。

Value

Asset Thinking / Win-Win / Operational Excellence / Capital Efficiency

資本効率と再現性を最優先し、標準化（SOP・教育・KPI）で拡大します。
